# Story: Player Controls

As a player, I want precise left/right control on desktop and mobile so that avoiding spikes feels fair and responsive.

## Scope

- keyboard input
- pointer/touch input
- player bounds
- movement tuning
- control feel validation

## Acceptance Criteria

- `ArrowLeft`, `ArrowRight`, `A`, and `D` move the bee.
- Dragging or touching the canvas moves the bee toward the pointer.
- The bee cannot leave the game bounds.
- Movement feels responsive without visible lag.
