# Task: Add Keyboard Controls [done]

## Goal

Allow desktop players to move the bee with keyboard input.

## Scope

- Support `ArrowLeft`, `ArrowRight`, `A`, and `D`.
- Track key down and key up.
- Clean up event listeners.

## Acceptance Criteria

- Holding left moves left.
- Holding right moves right.
- Releasing keys stops movement.
- Event listeners are removed on component unmount.

Done in current MVP.
