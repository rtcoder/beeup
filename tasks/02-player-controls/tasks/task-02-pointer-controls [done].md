# Task: Add Pointer And Touch Controls [done]

## Goal

Allow mobile players to move the bee by dragging or touching the canvas.

## Scope

- Use pointer events.
- Convert screen coordinates to fixed game coordinates.
- Smooth movement toward `targetX`.
- Disable browser scrolling on canvas interaction.

## Acceptance Criteria

- Dragging on the canvas moves the bee.
- Touching the canvas does not scroll the page.
- Pointer release clears the target.
- Controls work with responsive canvas scaling.

Done in current MVP.
