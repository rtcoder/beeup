# Task: Power-Ups [done]

## Story

Objects And Spawning

## Goal

Add readable shield and magnet pickups to the lane spawning system without blocking all safe lanes.

## Acceptance Criteria

- Shield and magnet power-ups can spawn after the safe start window.
- Power-ups use empty lanes from the current row.
- Power-ups are visually distinct from honey and spikes.
- Picking up a shield grants one spike absorb.
- Picking up a magnet temporarily pulls nearby honey toward the bee.
