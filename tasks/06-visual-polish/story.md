# Story: Visual Polish

As a player, I want Bee Up! to look charming, readable, and lightweight inside a Reddit post.

## Scope

- pastel background
- clouds/parallax
- simple bee animation
- honey and spike readability
- UI polish
- responsive presentation

## Acceptance Criteria

- The game has a coherent pastel arcade style.
- Entities are readable at speed.
- UI is minimal and does not cover gameplay.
- The game looks good on mobile and desktop widths.
