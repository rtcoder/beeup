# Story: Playable Core Loop

As a Reddit user, I want to open the post, start the game, play immediately, lose, and restart quickly so that Bee Up! feels like a short arcade session.

## Scope

- React/Vite app shell
- menu, playing, and game over states
- canvas game loop
- restart flow
- basic responsive layout

## Acceptance Criteria

- The app builds and runs locally.
- The user can start the game from a menu.
- The canvas renders a non-empty scene.
- Game over freezes gameplay and shows final score.
- Restart fully resets the game state.
