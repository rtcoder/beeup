# Story: Devvit Integration

As a developer, I want the game packaged as a Reddit Devvit Web custom post so that it can run inside Reddit.

## Scope

- Devvit project structure
- `devvit.json`
- custom post entrypoint
- create post action
- local playtest command
- Reddit iframe verification

## Acceptance Criteria

- The game can be launched through Devvit playtest.
- A custom post entrypoint points to the React client.
- A post creation flow can submit Bee Up! as an interactive post.
- Existing local Vite workflow remains useful for fast iteration.
