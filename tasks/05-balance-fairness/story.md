# Story: Balance And Fairness

As a player, I want the game to become harder over time while still feeling possible and readable.

## Scope

- difficulty ramp
- world speed scaling
- spawn interval scaling
- safe start
- impossible-pattern prevention
- playtesting balance values

## Acceptance Criteria

- First seconds are easy.
- After about 45 seconds the game is noticeably faster.
- Rows do not create impossible walls.
- Consecutive spike rows remain readable.
